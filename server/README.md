# my-contacts
Contact management API
<br />
<br />
<h2>Techs used:</h2>
<ul>
  <li>Node.js</li> 
  <li>Express</li>
  <li>Postgres (BD)</li>
</ul>

<h2>Requirements:</h2>
<ul>
  <li>Node >= 12</li> 
  <li>Docker</li>
  <li>Postgres image installed on Docker</li>
</ul>

<h2>To run the API on localhost:</h2>

<code>yarn dev</code>
